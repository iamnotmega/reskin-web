The Theme Installer allows you to install .reskin files created by the Theme Bundler manually. Here's how:

<Tooltip
  src={homeImg.src}
  alt="Reskin app homepage screenshot"
  tooltips={[
    { x: 2, y: -2, text: "Open the side navigation." }
  ]}
/>

<Tooltip
  src={sidenavImg.src}
  alt="Reskin app side navigation screenshot"
  tooltips={[
    { x: 30, y: 70, text: "Press the Theme Installer button." }
  ]}
/>

Once on the installer page, drag the .reskin file into the selection area or click on it to select the file manually.

When you have uploaded the .reskin file, press the Install Theme button.

You have just installed a theme from a bundled .reskin file onto your system!
